## 已实现功能 / Implemented Features

- [x] 图表显示
- [x] JSON导入/导出 (符合[UIGF统一可交换祈愿记录标准](https://github.com/DGP-Studio/Snap.Genshin/wiki/StandardFormat))

- [x] 当期UP池显示
